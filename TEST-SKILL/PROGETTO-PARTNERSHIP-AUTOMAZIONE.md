## EXECUTIVE SUMMARY

### Il Problema

Ogni centro assistenza compressori in Italia lavora cosi':
- Il tecnico riceve una chiamata → segna su carta o WhatsApp
- Va dal cliente → scopre che manca il pezzo di ricambio
- Torna in sede → ordina il ricambio → aspetta 3-5 giorni
- Torna dal cliente → ripara
- **Risultato:** 1 intervento = 4 spostamenti, 5-7 giorni di attesa, cliente insoddisfatto

### La Soluzione

Un **sistema web unico** che fa tre cose:

1. **SITO pubblico** — genera contatti e visibilita' (SEO automatizzato)
2. **GESTIONALE interno** — organizza tecnici, manutenzioni, scadenze in tempo reale
3. **MAGAZZINO intelligente** — traccia ricambi, prevede esaurimenti, suggerisce ordini

**Tutto in un'unica piattaforma.** Niente software da installare, funziona da telefono.

### Risultati Attesi

| Prima | Dopo |
|---|---|
| 1 intervento = 5-7 giorni | 1 intervento = 24-48 ore |
| Tecnico fa 3-4 interventi/giorno | Tecnico fa 6-8 interventi/giorno |
| Ricambi: "vado a vedere se ce l'ho" | Ricambi: il sistema sa cosa serve prima |
| Manutenzioni: il cliente chiama quando si rompe | Manutenzioni: il sistema avvisa prima che si rompa |
| Nessuna visibilita' online | Pagina 1 Google per zona |

#### Il Problema dei Ricambi Oggi

```
Tecnico parte → arriva dal cliente → manca il filtro
→ torna in sede → non c'e' neanche in magazzino
→ ordina al fornitore → aspetta 3 giorni
→ torna dal cliente → ripara
→ TOTALE: 5-7 giorni, 4 viaggi, cliente arrabbiato
```

#### Come lo Risolve il Sistema

```
Il sistema SA che:
  - Compressore GA30 di Acme Srl ha 4.000 ore
  - A 5.000 ore SERVE: filtro aria + filtro olio + cinghia
  - In magazzino HAI: filtro aria (2 pz), filtro olio (0 pz)
  - MANCA: filtro olio + cinghia

IL SISTEMA AUTOMATICAMENTE:
  ✓ 30 giorni prima → avvisa: "Ordina filtro olio e cinghia per GA30"
  ✓ 15 giorni prima → alert: "⚠ Manutenzione GA30 tra 2 settimane, ricambi mancanti!"
  ✓ Quando ordini → aggiorna magazzino
  ✓ Quando arriva il pezzo → "✓ Puoi programmare intervento Acme Srl"

RISULTATO: 1 solo viaggio, tutto pronto, cliente soddisfatto
```

| **Previsione consumi** | In base alle ore macchina dei clienti, il sistema PREVEDE cosa serviira' |
| **Lista ordini suggerita** | Ogni lunedi' → email con "Questa settimana dovresti ordinare..." |
