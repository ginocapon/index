Configurazione Gmail API

Step 1 — Crea progetto Google Cloud

  1. Vai su console.cloud.google.com
  2. Crea un nuovo progetto (es. "Righetto Admin")
  3. Dal menu, vai su API e servizi > Libreria
  4. Cerca "Gmail API" e abilitala

Step 2 — Crea credenziali OAuth2

  1. Vai su API e servizi > Credenziali
  2. Clicca + Crea Credenziali > ID client OAuth
  3. Tipo: Applicazione web
  4. Origini JavaScript autorizzate: aggiungi il tuo dominio (e http://localhost per test)
  5. Copia il Client ID

Step 3 — Configura la schermata di consenso

  1. Vai su API e servizi > Schermata di consenso OAuth
  2. Tipo utente: Esterno (poi aggiungi il tuo utente come tester)
  3. Compila nome app, email, ecc.
  4. Aggiungi scope: gmail.readonly, gmail.send, gmail.modify

Come configurare le API

Facebook + Instagram:

  1. Vai su developers.facebook.com
  2. Crea un'app di tipo "Business"
  3. Aggiungi il prodotto "Facebook Login"
  4. In Strumenti > Esplora API Graph, genera un Token di Pagina
  5. Per il token di lunga durata: usa il debug token e estendilo
  6. Il Page ID lo trovi nelle Impostazioni della Pagina Facebook
  7. L'Instagram Business Account ID: vai su API Graph Explorer, chiama /me/accounts poi /{page-id}?fields=instagram_business_account

Google Business Profile:

  1. Usa lo stesso Google Client ID della Posta
  2. Nella Google Cloud Console, cerca e abilita queste API:
    • My Business Account Management API — per trovare il tuo account
    • My Business Business Information API — per leggere le sedi
  3. Se dopo l'abilitazione la quota risulta 0, richiedi l'accesso alle API di Business Profile tramite il modulo Google
  4. Verifica che il periodo di prova Google Cloud non sia scaduto (l'upgrade è gratuito)
  5. Accedi con l'account Google proprietario/gestore della scheda Business
  6. Clicca "Connetti Google Business" qui sopra per autorizzare

TikTok:

  1. La pubblicazione diretta non e' supportata
  2. Genera l'immagine dell'immobile e caricala manualmente su TikTok
