Configurazione Gmail API

Step 1 — Crea progetto Google Cloud

  1. Vai su console.cloud.google.com
  2. Crea un nuovo progetto (es. "Righetto Admin")
  3. Dal menu, vai su API e servizi > Libreria
  4. Cerca "Gmail API" e abilitala

Step 2 — Crea credenziali OAuth2

  1. Vai su API e servizi > Credenziali
  2. Clicca + Crea Credenziali > ID client OAuth
  3. Tipo: Applicazione web
  4. Origini JavaScript autorizzate: aggiungi il tuo dominio (e http://localhost per test)
  5. Copia il Client ID

Step 3 — Configura la schermata di consenso

  1. Vai su API e servizi > Schermata di consenso OAuth
  2. Tipo utente: Esterno (poi aggiungi il tuo utente come tester)
  3. Compila nome app, email, ecc.
  4. Aggiungi scope: gmail.readonly, gmail.send, gmail.modify
